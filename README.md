## 🚀 如何在微信上运行

### 第一步：注册微信小程序账号

1. 打开 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击右上角「立即注册」→ 选择「小程序」
3. 用你的邮箱注册（建议用常用邮箱）
4. 选择「个人」主体类型（大学生选个人即可）
5. 填写身份信息完成注册
6. 登录后，在左侧菜单「开发」→「开发管理」→「开发设置」
7. 找到你的 **AppID**，复制下来

### 第二步：下载微信开发者工具

1. 打开 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 选择 **Windows 稳定版** 下载安装
3. 安装完成后打开

### 第三步：导入项目

1. 打开微信开发者工具
2. 点击「+」或「导入项目」
3. 填写信息：
   - **项目名称**：爱心减重计划
   - **目录**：选择 `e:\moxingdaima\xiaochengxu` 文件夹
   - **AppID**：粘贴你在第一步获取的 AppID
   - **开发模式**：小程序
4. 点击「确定」
5. 项目会在模拟器中自动编译运行

### 第四步：在手机上预览

1. 在开发者工具顶部工具栏点击「预览」
2. 会生成一个二维码
3. 用微信扫描二维码
4. 手机上就能看到小程序了！

## 👫 如何分享给女朋友使用

微信小程序正式发布需要审核，但**测试阶段**可以通过以下方式让女朋友使用：

### 方法一：添加为体验成员（推荐）

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 左侧菜单「管理」→「成员管理」
3. 点击「体验成员」→「添加成员」
4. 输入女朋友的**微信号**（不是手机号，是微信ID）
5. 她会在微信收到邀请，确认后即可使用
6. 之后她打开微信 → 发现 → 小程序 → 搜索你的小程序名即可

### 方法二：扫码预览（临时）

1. 开发者工具点击「预览」生成二维码
2. 把二维码截图发给女朋友
3. 她用微信扫码即可打开
4. ⚠️ 注意：这个方法有时间限制，二维码过期需要重新生成

### 方法三：上传为体验版

1. 开发者工具点击「上传」
2. 版本号填 `1.0.0`，描述填「初始版本」
3. 上传成功后，在微信公众平台 → 「管理」→「版本管理」
4. 在「开发版本」中点击「选为体验版」
5. 体验版可以被所有体验成员使用，更稳定

```json
{
  "appid": "wx你的AppID"
}
```

## ⚠️ 注意事项

1. **AppID 不要泄露**：AppID 可以公开，但 AppSecret 绝对不能上传到 GitHub
2. **`.gitignore` 建议**：不要把敏感配置上传到公开仓库
3. **正式发布**：如果想让更多人使用，需要在微信公众平台提交审核
4. **审核注意**：个人主体的小程序功能有限制，但本项目的功能都在允许范围内
5. **清真饮食**：所有默认食物数据符合清真标准，自定义食物请注意筛选
