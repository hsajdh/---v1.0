# 爱心减重计划 · 微信小程序

这是一个给女朋友定制的减肥/变美监督小程序。新版加入了“房间号 + 统一密码 + 双角色”流程：两个人输入同一个房间号和密码进入同一计划空间，再选择“监督者”或“变美者”。

## 新版核心功能

| 模块 | 功能 |
| --- | --- |
| 房间入口 | 首屏输入房间号、统一密码，并选择“监督者 / 变美者”。首次进入会自动创建房间。 |
| 监督者管理台 | 查看体重、积分、打卡天数、待处理申请；管理自定义食谱、运动、奖励、活动挑战；手动加减积分；处理申请。 |
| 变美者首页 | 查看减重进度、饮水、热量、运动、积分、活动挑战、体重趋势和推荐饮食。 |
| 每日打卡 | 变美者记录体重、运动、饮水和照片；监督者只查看打卡记录。 |
| 食谱库 | 内置校园清真减脂食物库；监督者可以添加/删除自定义食谱；变美者可以浏览并记录饮食。 |
| 奖励中心 | 变美者用积分兑换奖励；监督者可查看奖励池，并在管理台添加自定义奖励。 |
| 申请中心 | 变美者可以申请想要的食谱、奖励、活动或其他愿望；监督者可同意或拒绝。 |
| 活动挑战 | 监督者创建活动，设置积分；变美者完成活动后领取积分。 |

## 角色说明

### 监督者

- 进入后默认跳到“我的”页，也就是管理台。
- 可以查看变美者的打卡历史、体重变化、积分流水和申请。
- 可以添加自定义运动、奖励、活动。
- 可以进入食谱库添加自定义食物。
- 可以手动给变美者加分或扣分。
- 可以处理变美者提交的申请。

### 变美者

- 进入后默认跳到首页。
- 可以每日打卡、记录饮食、完成活动挑战、兑换奖励。
- 可以在“我的”页修改体重目标。
- 可以提交食谱、奖励、活动或其他愿望申请。

## 房间与数据

当前版本是本地存储房间版：

- 房间数据按 `房间号` 隔离，保存在微信小程序本地缓存中。
- 同一个设备或开发者工具里可以切换“监督者 / 变美者”查看完整流程。
- 如果要让两台不同手机实时共享同一个房间，需要再接入微信云开发数据库或自己的后端服务。

房间内保存的数据包括：

- 用户体重目标、今日数据、历史打卡
- 当前积分、积分流水、已兑换奖励
- 自定义食谱、自定义运动、自定义奖励
- 活动挑战、活动完成记录
- 申请记录和处理状态

## 如何运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择：

```text
E:\moxingdaima\xiaochengxu
```

4. 在 `project.config.json` 中把 `appid` 改成你自己的微信小程序 AppID。
5. 编译运行后，首屏会进入“专属变美房间”。
6. 输入房间号和统一密码，选择身份即可进入。

## 推荐测试流程

1. 输入房间号 `love520`，密码自定，选择“监督者”进入。
2. 在“我的”页添加一个奖励、一个活动，并给变美者加一些积分。
3. 点击“切换身份”变成“变美者”。
4. 去首页完成活动，去打卡页提交今日打卡。
5. 在“我的”页提交一个奖励或食谱申请。
6. 再切回“监督者”，处理申请并查看打卡记录。

## 注意事项

- 本项目不包含医疗建议，减重目标请以健康、安全为第一原则。
- 默认食谱按校园、清真、无需烹饪的场景设计，自定义食谱请自行判断是否适合。
- 本地缓存被清理后，当前设备上的房间数据会丢失。
- 正式给两台手机长期使用时，建议接入微信云开发，这样房间数据才能真正跨设备同步。
